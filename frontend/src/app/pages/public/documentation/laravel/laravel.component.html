Acerca do Laravel

Começamos bem, com pouca coisa nesta página.

Um dos nossos membros diz ao seus alunos que os sistemas são com as cebolas, quando cortas:

  • Vê-mos os aros
  • Dependem dos adjacentes
  • ...
  • Fazem chorar

Acima de tudo, é necessário compreender que estamos a construir uma aplicação descentralizada (o máximo possível). O Laravel é uma framework com provas dadas, desenhada para aplicações de grande dimensão, podendo com o Blade e RSS, criar aplicações Web. Contudo, deixámos o front-end para o Angular, e optimizámos esta camada não só para uma gestão de informação, mas com a capacidade de interagir com outros serviços sem perder a eficiência de quem navega no site. Veja o Laravel como a camada dos dados.

A nossa equipa terminou o Metronic, estamos a importá-lo no Angular, e estamos da fase das primeiras transacções entre as frameworks. Com a evolução dos trabalhos, esta será a página que irá crescer mais nas primeiras fases. contudo a API que disponibilizamos, tem de ser consultada em https://api.bewhy.org/ .

Futuramente, esta página será actualizada com mais conteúdos, sobretudo sobre os segredos imbuídos na template.
